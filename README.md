# manyi-react-admin
> manyi-cli 创建的React项目,该模板项目还在完善中...

## 项目环境
* node version 为16+
* 包管理工具使用yarn
* React 18 已不再支持IE浏览器。如需兼容IE，请选择低版本，另外搭建项目~
* 本项目搭建时，使用为淘宝镜像，如您使用其他镜像，请确认服务器上含有所需依赖

## 下载依赖
```
yarn install / yarn
```

## 启动开发环境
```
yarn run dev / yarn dev / yarn start
```

## 构建生产环境
```
yarn run build
```
